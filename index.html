Identity Document Check

Verify the structure before you trust the document.

Upload a PAN, Aadhar, or Passport image. Verum reads it with OCR, identifies the document type, and checks the ID number against its official format rules — catching obvious fakes, typos, and corrupted scans before they go any further.

What "verified" means here: this checks document format and checksum rules only — it does not connect to UIDAI, NSDL, or Passport Seva, since no public API for that exists. A passed check means the document is structurally valid, not officially confirmed by a government registry.
